## Problem Description

### Problem Statement

Given an array nums, return the first element that appears more than once. If no duplicate exists, return -1.

Example 1:

```text
Input: nums = [2,5,1,2,3,5,1,2,4]
Output: 2
Explanation: 2 is the first value encountered again.
```

Example 2:

```text
Input: nums = [1,2,3,4]
Output: -1
```

### Constraints:

•1 <= nums.length <= 10⁵

•1 <= nums[i] <= 10⁵

### Requirements

- Time complexity: O(n)

- Space complexity: O(n)

- Use hash set to track seen elements
