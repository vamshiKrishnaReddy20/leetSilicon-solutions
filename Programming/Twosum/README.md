### Problem Statement

Given an array of integers nums and an integer target, return true if there exist two distinct elements whose sum equals target.

Example 1:

```text
Input: nums = [2,7,11,15], target = 9
Output: true
Explanation: 2 + 7 = 9
```

Example 2:

```text
Input: nums = [1,2,3], target = 7
Output: false
```
