### Problem Statement

Given an integer array nums, return the maximum and minimum elements.

Example 1:

```text
Input: nums = [5,2,8,1,9]
Output: 9 1
```

Example 2:

```text
Input: nums = [42]
Output: 42 42
```

### Constraints:

•1 <= nums.length <= 10⁵

•-10⁹ <= nums[i] <= 10⁹
