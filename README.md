# LeetSilicon Solutions 🚀

Welcome to my [LeetSilicon](https://leetsilicon.com/) solutions repository! This repo contains my personal implementations and solutions to hardware engineering interview problems, tracking my preparation journey.

## 📚 About
LeetSilicon is an interview preparation platform tailored for hardware engineers. This repository serves as a centralized place for my code, notes, and testbenches for problems covering the following core domains:

* **Computer Architecture:** Cache, pipelining, branch prediction, and memory hierarchy.
* **RTL Design:** SystemVerilog, FSMs, Clock Domain Crossing (CDC), and timing.
* **Design Verification:** UVM concepts, assertions, coverage, and testbench creation.
* **Programming:** C++, Python, algorithms, and data structures.
* **Debug Code:** RTL debugging, GTKWave traces, and bug-fix challenges.

## 🗂️ Repository Structure

To keep things organized, the solutions are categorized by domain:
